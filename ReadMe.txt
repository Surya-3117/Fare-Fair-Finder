1. Create new ".env" file and type the following:

GOOGLE_MAPS_API_KEY="YOUR_API_KEY"
MONGO_URI=mongodb://localhost:27017/FareFairFinder

2.  Install Dependencies.
	npm install

3. Start MongoDB and make sure it is running.

4. Start the application using,
	node server.js

5. Visit your localhost port using browser to access the application.
